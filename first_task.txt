# Шпаргалка по Bash:

## Команды
Отобразить текущий каталог:
```
pwd
```
Создание файла:
```
touch test.txt
```

Создать каталог директорий:
```
mkdir -p .../.../...
```

Копирование:
```
cp что куда
```

Вырезать:
```
mv
```
Вывести содержимое файла:
```
cat
```
Удаление файла, каталога, непустого каталога
```
rm, rmdir, rm -r
```
